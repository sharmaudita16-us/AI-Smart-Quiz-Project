# 🤖 AI Smart Quiz

## 📌 Description
AI Smart Quiz is a simple web-based quiz application that provides questions based on selected topics. 
It uses predefined question sets and returns relevant questions when a user searches for a topic like Java, DBMS, OS, or CN.

## 🚀 Features
- Topic-based quiz (Java, DBMS, OS, CN)
- Predefined questions and answers
- Multiple choice questions (MCQs)
- Instant question generation based on input
- Simple and clean UI

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js (Express)
- API: Custom REST API (`/api/quiz`)

## ⚙️ How it works
1. User enters a topic (like "Java")
2. Frontend sends request to backend (`/api/quiz`)
3. Backend checks topic using conditions
4. Returns related questions and options
5. Questions are displayed on screen
